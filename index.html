O que é HTML?

Com o surgimento da web, era necessário criar uma linguagem que fosse entendida por meios de acesso diferentes. Para tanto, Tim Berners-Lee desenvolveu o HTML, com a proposta de suprir essa necessidade. Somente na década de 1990, quando o Mosaic – browser desenvolvido por Marc Andreessen – se popularizou, o HTML ganhou força e foi adotado por outros desenvolvedores e fabricantes de browsers, compartilhando as mesmas convenções.
HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.
Para que você possa entender bem, o HTML é baseado no conceito de hipertexto, que são conjuntos de elementos ligados por conexões, que podem ser palavras, imagens, vídeos, áudio, documentos etc., que quando conectados, formam uma grande rede de informação. A conexão feita em um hipertexto é algo imprevisto que permite a comunicação de dados, organizando conhecimentos e guardando informações relacionadas.
Entre 1993 e 1995, surgiram novas versões (HTML+, HTML 2.0, HTML 3.0) onde foram propostas diversas mudanças para enriquecer as possibilidades da linguagem, mas ela ainda não era tratada como padrão. Apenas em 1997, o grupo de trabalho do W3C, trabalhou na versão 3.2, fazendo com que ela fosse tratada como prática comum.
Resumindo, e creio que você já tenha percebido, o HTML foi criado, e melhorado, para se tornar uma linguagem independente de plataformas ou meios de acesso, diminuindo custos e fazendo com que a web não ficasse limitada a uma base proprietária com formatos incompatíveis.
Em 2004, foi fundado o WHATWG (Web Hypertext Application Technology Working Group) por desenvolvedores de empresas como Mozilla, Apple e Opera, onde se iniciou o trabalho de escrever a nova versão do HTML, que seria chamada hoje de HTML5.

Por que aprender HTML?

Aprender HTML é importante para qualquer um que queira ter uma carreira no desenvolvimento web. Desenvolvedores talentosos estão sempre em demanda, sejam eles trabalhando para agências ou independentemente, e aprender HTML é a base desse conjunto de habilidades.

Tags HTML

Existem 142 tags de HTML diferentes que permitem que você crie elementos. Elas são compostas de elementos de nível de bloco e em linha.
Os elementos de nível de bloco abrangem toda a largura de uma página, iniciando uma nova linha dentro de um documento. Aqui estão algumas tags comuns de nível de bloco que você pode usar em seu site:

  • <header>: Define o cabeçalho de uma seção ou página.
  • <nav>: Indica uma seção de navegação.
  • <main>: Define o conteúdo principal de uma página.
  • <article>: Representa um conteúdo independente e autossuficiente.
  • <section>: Define uma seção genérica de um documento.
  • <aside>: Representa conteúdo relacionado, como barras laterais.
  • <footer>: Define o rodapé de uma seção ou página.
  • <h1> a <h6>: Cabeçalhos de diferentes níveis.
  • <p>: Define um parágrafo de texto.
  • <img>: Cria uma imagem.
  • <video>: Permite a incorporação de vídeos.
  • <audio>: Insere arquivos de áudio reproduzíveis na página.
  • <ul>: Cria uma lista não ordenada.
  • <ol>: Cria uma lista ordenada.
  • <li>: Define um item de lista.
  • <table>: Cria uma tabela.
  • <form>: Define um formulário para entrada de dados.
  • <input>: Cria um campo de entrada, como caixa de texto, botão, etc.
  • <textarea>: Cria uma área de texto multilinha.
  • <label>: Rótulo para um elemento de formulário.
  • <select>: Cria um menu suspenso.
  • <canvas>: Permite a criação de gráficos e animações.
  • <svg>: Insere gráficos vetoriais escaláveis.
  • <div>: Define uma divisão genérica ou contêiner.
  • <span>: Define uma divisão em linha.
